    About

    Link Cleaner is a privacy-first browser extension that strips 88+ tracking parameters from URLs before navigation. Zero data collection. Open source (MIT).

    • 🌐 GitHub: afiqandico13/link-cleaner
    • 📦 Database: 88 tracked params + prefix patterns
    • 🔧 Manifest V3 (Chrome, Edge, Brave, Firefox 109+)
    • 📝 License: MIT
    • 👤 Made with 🛡️ by @afiqandico13

    How it works

    Content scripts run on every page. They intercept link clicks, rewrite anchor href attributes, and watch for new links via MutationObserver. When you click a link, the tracking params have already been stripped — you navigate to a clean URL.

    Settings are stored locally (chrome.storage.local) and sync live to all open tabs. No data leaves your browser.
